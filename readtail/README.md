# ReadTail - 阅读尾巴教育应用

一个互动式儿童阅读教育应用，包含阅读、测验、宠物养成等功能。

## 功能模块

### ✅ 已完成模块

1. **首页** (`/`)
   - 用户信息展示
   - 宠物睡眠状态展示
   - 今日阅读进度
   - 快速阅读、真题检测、阶段总结入口

2. **书库选择** (`/books`)
   - 推荐书籍/你的书籍标签切换
   - 3D书籍卡片旋转展示
   - 宠物互动提示

3. **宠物解锁** (`/pet-unlock`)
   - 展示三个宠物皮肤（已解锁/待解锁）
   - 宠物对话气泡
   - 继续阅读按钮

4. **阅读区域** (`/reading`)
   - 左侧：书籍封面、章节导航、音频播放
   - 中央：文本内容展示（支持插图）
   - 右侧：学霸笔记面板（可展开/收起）
   - 左下角：宠物每日目标提示（点击开始阅读后消失）
   - 文本高亮、收藏/不懂标记功能

5. **摄像头模式** (`/camera`)
   - 实时摄像头画面
   - 模拟书籍识别（自动识别为《偷影子的人》）
   - 右侧学霸笔记展示

6. **互动答题** (`/quiz`)
   - 进度条显示
   - 2x2网格选项布局
   - 宠物实时反馈（答对/答错）
   - 自动跳转下一题

7. **答题报告** (`/quiz-report`)
   - 圆形进度环（阅读度、正确度）
   - 返回首页/再测一次按钮

8. **总结页面** (`/summary`)
   - 标记为"正在开发"
   - 预告即将推出的功能

### 🎨 设计特点

- 温暖的米黄色背景渐变
- 圆角卡片设计
- 流畅的过渡动画
- 宠物互动反馈
- 底部圆形图标导航栏

## 技术栈

- **前端框架**: React 19
- **构建工具**: Vite 5.4
- **路由**: React Router DOM 7
- **样式**: 原生 CSS（无UI框架）

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 浏览器访问

开发服务器启动后访问: http://localhost:3000

## 注意事项

1. 需要 Node.js 版本 >= 14.18.0
2. 摄像头功能需要 HTTPS 或 localhost 环境
3. 音频播放需要用户交互后才能自动播放
4. 部分图片资源较大，首次加载可能需要时间
